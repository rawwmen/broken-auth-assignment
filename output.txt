===== 1. LOGIN =====
POST /auth/login

Request Body:
{
  "email": "user@test.com",
  "password": "123"
}

Response:
{
  "message": "OTP sent",
  "loginSessionId": "s3u0d"
}


===== 2. VERIFY OTP =====
POST /auth/verify-otp

Request Body:
{
  "loginSessionId": "s3u0d",
  "otp": "434655"
}

Response:
{
  "message": "OTP verified",
  "sessionId": "s3u0d"
}
